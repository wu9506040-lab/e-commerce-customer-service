# Sprint 21 知识库种子补完 · 6 个 JSON · 33 条真实政策数据源（2026-07-21）

> **目的**：CLAUDE.md §1 项目身份"电商智能客服 Agent"服务对象为淘宝/京东/拼多多。Sprint 18-20 已落地 12 个 Service（业务广度 85%），但 RAG 知识库**之前是手工编的虚构数据**（"智选科技官方旗舰店"），无法回答"拼多多 2025 年仅退款怎么改 / 京东 30 天保价怎么申请 / 平台介入要几天"等真实问题。
>
> S21 目标：**用真实公开政策数据替换虚构 KB**，覆盖 S20 新增的 SupplementRuleService（5 方法）+ DisputeService（3 方法）+ 三家平台对比。
>
> **设计权衡**：按 .gitignore 设计意图，KB 数据保留本地不上公开仓库。本文档记录数据源 URL + 内容摘要 + 重建命令——任何协作者可按本文档重新获取数据。
